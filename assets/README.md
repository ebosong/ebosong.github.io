# Assets Placeholder Guide

Add your personal files here:

- `favicon.ico` - Browser tab icon.
- `CV.pdf` - Your curriculum vitae (linked by "View CV" and Contact section).
- `profile.jpg` - Optional profile photo (you can replace the placeholder block in `index.html`).
- `og-image.png` - Social sharing preview image for Open Graph tags.

If a file is missing, the site still loads, but related links/icons may not appear as expected.
